# Escalation Scenarios

Work through each scenario below. For each one:
1. **Classify** the issue type
2. **Assess** the severity level (Critical/High/Medium/Low)
3. **Determine** who to escalate to
4. **Decide** on the timeline
5. **Draft** a brief escalation summary (2-3 sentences)

Write your answers before checking the solutions.
