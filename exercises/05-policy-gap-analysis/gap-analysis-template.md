# Gap Analysis Template

## Instructions

For each control area:
1. Identify the relevant policy/control (if any) from the mock policies
2. Assess the current status: **Full / Partial / None**
3. Identify specific gaps
4. Determine gap severity: **Critical / High / Medium / Low**
5. Recommend remediation actions
